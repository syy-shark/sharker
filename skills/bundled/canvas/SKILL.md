# Canvas

Prefer a **Canvas** artifact when the deliverable is:

- Data-heavy (billing, metrics, comparisons)
- Visual (charts, timelines, architecture maps)
- Interactive (filters, tabs, expandable sections)

Do not dump large markdown tables when a structured canvas layout is clearer.
