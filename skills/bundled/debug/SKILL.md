# Debug

1. Reproduce minimally
2. Form 1–2 hypotheses
3. Gather evidence (logs, stack, bisect)
4. Fix root cause, not symptoms
5. Add or run a test that would have caught it

Report: what you observed, what you tried, what fixed it.
