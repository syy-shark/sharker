# Git Commit

- **Subject**: imperative, ≤72 chars, explains *why* not *what*
- **Body**: context, trade-offs, test plan bullets
- One logical change per commit when possible
- Never commit secrets or `.env`

For PRs: Summary (2–3 bullets) + Test plan checklist.
