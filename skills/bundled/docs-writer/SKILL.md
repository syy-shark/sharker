# Docs Writer

- Lead with **what it is** and **how to run it**
- Use headings, tables, and code blocks for scanability
- Keep examples copy-pasteable
- Sync docs when behavior changes (same PR when possible)

Match the repo's existing doc tone and language (中文/English as appropriate).
