# Create Rule

Before writing a rule, confirm:

1. **Purpose** — what behavior should persist?
2. **Scope** — always apply, or file-specific globs?
3. **Format** — `.mdc` with frontmatter (`description`, `globs`, `alwaysApply`)

Keep rules short, actionable, and one concern per file.
